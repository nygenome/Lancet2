# Getting Started

Clone the repository from the [Lancet GitHub](https://github.com/nygenome/Lancet2)

# Prerequisites

* CMake >= 3.14.x
* C++ compiler with support for ISO C++17 standard 
* zlib, bzip2, liblzma, cURL and OpenSSL
